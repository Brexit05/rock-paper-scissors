# rock-paper-scissors

A simple Rock Paper Scissors game built with HTML and JavaScript.

## How to Play

- Open index.html in your browser
- Enter rock, paper, or scissors when prompted
- Play 5 rounds against the computer
- Highest score after 5 rounds wins
- If it's a tie, you can replay
